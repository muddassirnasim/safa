A premium, super-fast, very easy-to-use WhatsApp chat plugin with analytics and a beginner-friendly dashboard.

== Description ==
Smart WhatsApp Chat adds a floating WhatsApp chat button to your site with optional multi-agent routing and privacy-friendly analytics.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/smart-whatsapp-chat` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Open WhatsApp Chat in the admin menu to complete Quick Setup.
